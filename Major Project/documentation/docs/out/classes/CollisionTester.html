findNearestTerrainVertex

() Int private

Defined in CollisionTester.js:96

Need to find what height to position the player at. So need to find what terrain vertex they're nearest to.

Do this by passing in the player X and Z coordinates into the heightMap, to return the corresponding terrain vertex height value.

Get the nearest height from the heightMap, which is private, So call the getter method

Returns:

Int:

the height of the nearest vertex + 4, to position the camera at adding + 4 because otherwise the camera would be in the floor

Moves the player forwards/backwards depending on the direction they where moving when they collided

If direction === 1 Then player has collided moving forwards, so move the player backwards If direction === -1 Then player has collided moving backward, so move the player forwards

Parameters:

  • direction Int

    the direction to push them, push backwards = 1, push forwards = -1
